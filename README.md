# 🎓🤖 VivaPrep AI

> AI-Powered Viva & Exam Preparation Platform

VivaPrep AI is a full-stack AI-powered web platform designed to help students prepare smarter for viva exams and academic interviews. Users can upload PDFs, notes, study materials, or enter custom topics, and the system generates relevant viva questions and answers using AI.

---

## 🚀 Features

✅ Upload PDF / Notes / Study Materials  
✅ AI analyzes content intelligently  
✅ Generates Viva Questions & Answers  
✅ Topic-based custom question generation  
✅ Saved Reports System  
✅ Premium Dashboard UI  
✅ Interactive Mini-Games while generating output  
✅ Secure Authentication System  
✅ Cloud Database Integration  
✅ Fast & Responsive UI  

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- CSS3

### Backend
- Node.js
- Express.js

### Database / Auth

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/ashmittita10a-creator/VivaPrep_AI.git
cd VivaPrep_AI
```

### 2️⃣ Backend Setup

```bash
cd backend
npm install
node server.js
```

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create `.env` file in backend:

```env
SUPABASE_URL=your_url
SUPABASE_KEY=your_key
TOGETHER_API_KEY=your_key
GROQ_API_KEY=your_key
```
