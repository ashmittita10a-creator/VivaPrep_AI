# VivaPrep AI 🎓🤖

VivaPrep AI is an AI-powered exam preparation platform that generates viva questions and answers from PDFs, notes, study materials, or custom topics.

## Features

- Upload PDF / Notes
- AI Viva Question Generation
- Topic-based Questions
- Saved Reports
- Premium Dashboard UI
- Mini Games While Loading
- Secure Login Authentication

## Tech Stack

Frontend: React.js, Vite, CSS  
Backend: Node.js, Express.js  
Database: Supabase  
AI APIs: Together AI, Groq API
